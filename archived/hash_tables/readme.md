# Hash tables
Fast at access, insert, delete but unordered and can have collisions

![hash table](images/hash_table.jpg)
![hash table collision](images/hash_table_collision.png)

## BigO
Without collision
- insert O(1)
- lookup O(1)
- delete O(1)
- search O(1)

With collision
- insert O(n)
- lookup O(n)
- delete O(n)
- search O(n)
